# Gene Information

## Files

Information about genes in *Mycobacterium tuberculosis* (Mtb).

- Mycobacterium_tuberculosis_H37Rv_txt_v5.tsv: Information about Mtb genes from
  [Mycobrowser](https://mycobrowser.epfl.ch/)
- bosch_vi.xlsx: Essentiality and vulnerability information retrieved from from
  [Bosch et al. 2021](https://pmc.ncbi.nlm.nih.gov/articles/PMC8382161/), with
  the essentiality data coming from Dejesus et al. 2017
